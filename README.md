# کفش‌پلاس

فروشگاه آنلاین کفش به زبان فارسی با Next.js و Tailwind CSS.

## ویژگی‌ها

- طراحی RTL کامل برای زبان فارسی
- فونت Vazirmatn
- صفحات: خانه، محصولات، جزئیات محصول، درباره ما، تماس با ما
- کامپوننت‌های ماژولار و واکنش‌گرا
- بهینه‌سازی تصاویر با `next/image`

## شروع

```bash
npm install
npm run dev
```

سایت روی [http://localhost:3000](http://localhost:3000) در دسترس است.

## ساختار پروژه

```
app/
  page.tsx              # صفحه اصلی
  products/             # لیست و جزئیات محصولات
  about/                # درباره ما
  contact/              # تماس با ما
components/             # کامپوننت‌های UI
lib/                    # داده‌ها و ابزارها
```

## تکنولوژی‌ها

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
- Lucide React (آیکون‌ها)
